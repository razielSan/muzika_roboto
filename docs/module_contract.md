# Module Contract v1

## Required files
- router.py
- settings.py
- response.py

## Required directories
- childes (or configured separator)
- handlers
- services
- utils
- keyboards
- api
- fsm

## settings.py must define:
- SERVICE_NAME
- ROOT_PACKAGE
- NAME_FOR_TEMP_FOLDER
- NAME_FOR_LOG_FOLDER
- SHOW_IN_MAIN_MENU
- MENU_CALLBACK_TEXT
- MENU_CALLBACK_DATA
- MENU_REPLY_TEXT
